Start de Node.js-server.

In de map waar dit bestand staat ligt een launcher. Hij installeert dependencies (eenmalig), start de Node-server, en opent je browser op http://localhost:3000. Voor volledige HTTPS zet je Caddy ervoor — dat regelt het launcher-script automatisch.

Dubbelklik op Start Workshop.command Een Terminal opent. Eerste keer: npm install draait (~30 sec) en optioneel installeert het script Caddy voor HTTPS.
3
Wacht tot je browser opent. De Node-server start op poort 3000, Caddy zet er HTTPS voor op poort 8443 (als geïnstalleerd).
4
Klaar — je werkt nu via een eigen server. Live-sync loopt via WebSocket op je eigen machine, niet meer via publieke brokers. Sluit Terminal om alles te stoppen.

Een use case beschrijft hoe een persoon een systeem gebruikt om een specifiek doel te bereiken — inclusief de stappen, alternatieve scenario's en de voorwaarden. Het focust op het "wat" (het doel) in plaats van het "hoe" (de technische implementatie).
